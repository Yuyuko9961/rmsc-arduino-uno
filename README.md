## 本仓库为 rmsc-mod 项目的嵌入式部分  

- **开发环境:** VScode with PlatformIO
- **开发平台:** Atmel AVR (5.1.0) > Arduino Uno R3
- **硬件信息:**
    - **主芯片:** ATMEGA328P 16MHz, 2KB RAM, 31.50KB Flash
    - **OLDE屏幕:** SH1106, 128x64, I2C
    - **旋转编码器:** EC11
